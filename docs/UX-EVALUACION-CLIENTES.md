# Encuesta de evaluación UX/UI — GymApure

**Propósito:** Recopilar retroalimentación de clientes reales (cuentas de simulación) tras el test integral del sistema, para identificar problemas de usabilidad y priorizar correcciones de raíz.

**Instrucciones para el evaluador:**

1. Inicia sesión con tu cuenta de simulación (email proporcionado por el administrador).
2. Usa la app en tu **dispositivo habitual** (móvil recomendado).
3. Explora las secciones indicadas durante **15-20 minutos**.
4. Responde cada pregunta con honestidad — no hay respuestas correctas o incorrectas.

**Contraseña:** La misma para todas las cuentas de simulación (proporcionada por el admin del test).

## Procesamiento de resultados

1. **Recopilar** encuestas completadas (digital o impreso).
2. **Calcular** promedios por sección y por nivel (principiante/intermedio/avanzado).
3. **Identificar** patrones: problemas que aparecen en ≥3 evaluadores del mismo nivel.
4. **Priorizar** con matriz impacto/esfuerzo:
   - **P0:** Bloquea uso (ej. no puede completar entrenamiento)
   - **P1:** Alta fricción (ej. confusión recurrente en nutrición)
   - **P2:** Mejora deseable (ej. estética, microcopy)
5. **Documentar** en issues con etiqueta `ux-feedback`.
6. **Re-ejecutar** `npm run test:full-system -- --fast` tras correcciones.
